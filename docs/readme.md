# Arquitectura del Proyecto - DiDi Logistics Optimization

## 1. Objetivo
Optimizar rutas y tiempos de entrega utilizando análisis de eficiencia logística.

## 2. Fuentes de Datos
- Dataset principal: sgt_didi_delivery_efficiency_2026.csv
- Diccionario de variables: docs/diccionario_variables.md

## 3. Flujo de Procesamiento

1. Limpieza de datos
2. Validación de inconsistencias
3. Cálculo de métricas clave
4. Análisis de eficiencia
5. Visualización de resultados

## 4. Stack Tecnológico
- Excel (captura y estructuración)
- SQL (consultas y agregaciones)
- Git (control de versiones)

## 5. Supuestos y Limitaciones
- No se consideran variables climáticas
- Datos limitados a 24 días operativos
